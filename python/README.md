# EDInburgh Python Module

NOTE: this is just a stub for the Python module. It is not functional yet.

Uses [pyo3](https://github.com/PyO3/pyo3) to create a Python bindings.

## Installation

```shell
# virtualenv
python -m venv .venv
source .venv/bin/activate

# maturin
pip install maturin
```

## Build (Development)

```shell
maturin develop
```

```shell
python3 example.py
```
